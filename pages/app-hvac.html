HVAC 모니터링

환경 모니터링 · 적용 센서: 차압 · 온도

공조 설비의 온도·차압·필터 상태를 추적해 운영 효율과 유지보수 시점을 관리합니다.

온도·습도·누수·공기질 등 실내외 환경의 모든 변수를 통합 추적.

01 · 고객 프로필

공조 효율과 유지보수 상태를 관리해야 하는 상업용 빌딩·캠퍼스·병원·데이터센터.

공조 설비는 온도·차압·필터 상태 관리가 핵심입니다. HVAC 이상은 에너지 낭비·쾌적성 저하·정비비 증가로 이어지므로, 상태를 실시간 확인하고 이상 패턴을 조기에 잡습니다.

02 · 현재의 문제점

BMS만으로는 필터 막힘·덕트 누설·차압 이상 같은 세부 문제를 빠르게 진단하기 어렵습니다.

  • BMS 감지 한계 — 필터 막힘·국부 차압 이상까지 세밀 감지 곤란
  • 에너지 낭비 — 공조 이상이 과도한 냉난방·전력 사용 유발
  • 정비 시점 불명확 — 데이터 없으면 과잉·지연 정비 발생
  • 쾌적성 저하 — 온도·환기 불안정 시 불편·민원 증가

03 · Monnit 솔루션

온도·차압 센서로 공조 상태를 감시하고 iMonnit에서 이상 징후·이력 확인.

온도·차압 센서로 공조 온도·덕트 상태·필터 전후 압력을 상시 감시합니다. iMonnit이 설비별 추이를 시각화하고 기준 이탈 시 알려, 전체를 보는 BMS에 세부 지점을 보완적으로 더합니다.

구성 센서: 차압 · 덕트 온도 · 온도 · 습도 · 전류 · 전압 · 개폐 감지 · 건접점

04 · 제안 단계

단일 빌딩 파일럿 우선 → 캠퍼스 전체 확장

  • PHASE 1 기준 빌딩 선정 (1주) — 에너지 효율 진단 가능한 대표 빌딩 선정.
  • PHASE 2 센서 배치 (1주) — 필터·덕트·펌프 등 진단 포인트 12–20개 배치.
  • PHASE 3 3개월 데이터 분석 (3개월) — 이전 BMS 데이터와 비교해 비효율 진단.
  • PHASE 4 캠퍼스 확장 (2–4개월) — 검증된 구성으로 다른 빌딩 일괄 배포.

05 · 예상 ROI

HVAC 운영비 연 15~25% 절감 · 18개월 내 회수

  • 15~25%↓ — HVAC 운영비 절감
  • 30%↓ — 필터 교체 비용 감축
  • 18개월 이내 — 평균 ROI 회수 기간

도입 전: BMS 평균 데이터만 보유. 필터 교체는 시간 기반. 펌프 이상은 고장 후 인지.

도입 후: 필터 차압 정밀 추적. 데이터 기반 교체 주기. 펌프·팬 부하 패턴으로 정비 시점 예측.
