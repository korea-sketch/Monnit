Installation Guides

세 가지 온도 센서 활용: Low Temperature Sensor(ULT 냉동고용, 리드를 내부에 삽입, -200~0°C/-328~32°F를 ±0.5°C 정확도로 모니터링), Standard Temperature Sensor(-40~257°F/-40~125°C), ALTA Digital Temperature Sensor(버튼으로 LCD에 즉시 판독).

리드 삽입 방법:

  • A) 사전 천공 knockout이 있는 경우: 제조사 지침대로 뚫고 리드 삽입 후 실리콘 등으로 밀봉.
  • C) 직접 구멍 뚫기: 코일·핵심 부품이 없는 곳 확인(매뉴얼 참조, Monnit은 천공 손상 책임 없음) → 0.25"(6mm) 구멍 → 프로브 삽입·내부 고정 → grommet/실리콘 밀봉.
  • D) 도어 실(seal)을 통과: 문 닫을 때 리드가 눌리지 않는지 확인 → 도어 실에 0.25"(6mm) 구멍 → 리드 통과·내부 프로브 고정 → 누수 방지 밀봉.

Installation Guides

ALTA Wireless Propane Tank Level Monitor는 R3D(Remote-ready) 센서를 사전 설치된 R3D 탱크 게이지에 꽂아 프로판 레벨을 원격 추적합니다(레벨 감소 → 판독값 감소 → 잔량 %로 변환·무선 전송). Jr.·Sr. R3D 게이지 지원.

설치: 자석으로 6ft 리드가 게이지에 닿을 공간을 두고 탱크에 장착 → 리드의 검은 팁을 게이지 홈에 정렬해 밀어 넣음(딸깍 소리 = 성공).

토양 수분 센서 설치 방법

Installation Guides

센서 구성요소: A-저항성 그래뉼러 매트릭스·온도 소자, B-63인치 분리형 리드, C-산업용(IP65 NEMA) 하우징, D-On/off 스위치, E-무선 안테나.

준비물: PVC 파이프(1/2" Class 315 또는 3/4" CPVC SDR 11), ABS-PVC 전환 시멘트, 비닐 캡, 케이블 타이 3개. 공구: 1/8" 드릴, 쇠톱, 7/8" 로드/오거(연마성 토양은 1").

Wetting 과정(필수, 매설 전): 센서를 관개수에 하룻밤 담금. 항상 젖은 상태로 설치. 가능하면 며칠에 걸쳐 부분 침수(절반 이하, 30분)·건조 반복 후 WET 상태 설치 → 초기 관개 응답 개선.

물리적 설치: 7/8"(22mm) 로드/오거로 원하는 깊이의 구멍 → 흙+물 슬러리를 바닥에 채우고 센서를 단단히 밀어 넣어 "grout in"(표면 접촉 극대화) → 1/2" PVC를 센서 collar에 끼워 삽입(전환 시멘트로 용접 가능). 물이 리드를 타고 소자에 직접 닿지 않도록 리드 수평 구간(6~12")을 3~6" 다진 흙 아래 매설. PVC를 남기면 표면 흙 다짐·상단 캡 밀봉(표면수 침투 방지), 바닥에 작은 구멍 뚫어 갇힌 물 배수. 센서는 작물 활성 뿌리 영역에 위치. 깃발로 표시 권장.

거칠/자갈 토양: 막 손상 방지를 위해 큰 구멍(1~1.25") + 슬러리, 또는 "stepped" 도구(상부 큰 구멍, 하부 정확한 크기) 사용·공기 주머니 방지 다짐.

제거·보관: 청소·건조 후 깨끗한 곳에 무기한 보관(스위치 Off로 배터리 절약). 재설치 전 항상 재wetting.

ALTA Current Meter 구성·설치 방법

Installation Guides

ALTA Wireless AC·삼상 Current Meter는 변류기(CT)로 AC 시스템의 RMS 전류를 측정합니다. CT는 단상(AC) 또는 3선(hot, common, ground) 전원의 "hot" 선을 감쌉니다. ALTA 150·500A 단상·삼상 미터는 M12 5핀 커넥터 분리형 리드로 전기 패널에 쉽게 설치.

설정: Minimum/Maximum Threshold(Aware State 진입·보고), Averaging Interval(Heartbeat 내 평가 빈도(초), 평균하여 Average Current 산출), Accumulate(Off=단일 Heartbeat 소비량만, On=이전 측정에 계속 누적), Show Full Data(Off=누적만, On=평균·최대·최소 추가). 추가로 Scale(Amp-hours↔Power Watt/kWh, 시스템 전압 설정 — 미국 120VAC, EU 240VAC 등), Calibration.

설치: 게이트웨이 통신 확인 → CT가 전원선에 클립될 공간 두고 장착 → CT를 모니터링할 전원선에 부착(단일 통전 선만 감쌀 것, common+live 케이블을 감싸면 약 0.00A로 읽힘). 방향 화살표는 무관 → CT 리드를 M12 5핀 커넥터로 하우징에 연결(삼상은 1·2·3 포트에 각 상 연결).

Air Velocity 센서 구성·설치 가이드

Installation Guides

ALTA Wireless Air Velocity 센서는 두 포트 간 압력차를 측정하고 주변 온도·고도를 반영해 HVAC 기류 속도(m/s)를 산출합니다.

설정: Below/Above (CFM)(Aware State 진입), Altitude(해발 미터, 기본 0m, 정확도 향상), Show Temperature. Scale로 단위 변경(Mps/mph/kmph/knot/CFM/CMH/CMM).

설치: 게이트웨이 통신 확인 → 리드 끝이 측정 포트에 삽입될 공간 두고 장착 → 나사로 리드 끝을 포트에 고정. 센서 전면에서 볼 때 오른쪽 입구 포트가 양압(고압) 입력(리드 끝은 반대로 왼쪽이 고압). 오른쪽 압력이 크면 음압, 왼쪽이 크면 양압 판독. 온도·고도와 결합해 기류율 산출.

ALTA Humidity 센서 구성·설치 가이드

Installation Guides

ALTA Wireless Humidity 센서는 고정밀 실리콘 디지털 습도/온도 소자로 상대습도(%RH)·온도를 측정하며, iMonnit에서 이슬점·수분량·열지수·습구온도를 계산합니다.

설정: 습도 Below/Above(Aware State 진입), Aware State Buffer %, 온도 Below/Above, 온도 Aware State Buffer(예: 최대 90°·Hysteresis 1° → 90.0°에 Aware 진입, 89.0°까지 떨어져야 정상 복귀). Show Moisture Point(고도 필드), Show Dew Point, Show Heat Index, Show Wet Bulb.

Calibrate: 안정 환경에 두고 판독값 안정화 대기 → 관측-실제 차이를 Offset에 입력 후 Calibrate.

Scale: 화씨/섭씨.

설치: 게이트웨이 통신 확인 → 하우징 바닥 구멍·리드 위치로의 기류를 막지 않는 곳에 장착.

문제 해결: "Hardware Error"(센서 베이스가 감지 소자와 통신 불가) → Monnit 기술 지원 문의.

Dual Pulse Counter 구성·설치 가이드

Installation Guides

ALTA Wireless Dual Pulse Counter는 두 펄스 출력 장치에 연결해 각 장치의 펄스 수를 동시 기록합니다.

설정: Name, Heartbeat Interval, Overflow Limit(초, 초과 시 aware 플래그로 즉시 보고), Synchronize(Off=마지막 보고 1 Heartbeat 후 / On=UTC 시간 정렬, 게이트웨이당 10개 이하 권장), Detection Type(positive/negative/both edge), Pulse Noise Filter(Strong=초당 4회 미만, Medium=20회 미만, Weak=80회 미만, None), Accumulate(Off=Heartbeat 간 펄스 수 / On=계속 누적, 64비트 롤오버 / Reset=0으로), Overflow Count(Heartbeat 내 이 수 관찰 시 aware 즉시 보고, 통신 시 리셋).

Synchronize 예시 (Heartbeat 60분, 1:47 PM 보고): Off면 2:47·3:47 PM, On이면 2:00·3:00 PM.

설치: 게이트웨이 통신 확인 → 리드선이 접점에 닿을 공간 두고 장착 → 선을 단점에 연결.

최종 수정일: 2026년 1월 13일

공통 설정 참고 (이후 다수 센서에 반복): Aware State Buffer/Hysteresis는 임계값 근처에서 표준↔Aware 진동 방지(예: 최대 90°·버퍼 1° → 90.0°에 Aware, 89.0°까지 떨어져야 복귀). Synchronize Off=마지막 보고 1 Heartbeat 후 / On=UTC 시간 정렬(게이트웨이당 10개 이하 권장). Assessments per Heartbeat=Heartbeat 사이 임계값 확인 횟수.

ALTA Wireless Control Unit 구성·설치 가이드

Installation Guides

ALTA Control Unit은 iMonnit으로 개별 제어 가능한 두 릴레이를 가집니다(수동 또는 센서 알림으로 자동 On/Off). 15A는 래칭(latching) 릴레이로 전원 꺼져도 위치 유지(전원 손실 시 위치 유지 필요할 때 유용), 10A·30A는 비래칭으로 전원 꺼지면 원래 상태 복귀(정전 시 셧오프 페일세이프).

설치: 안테나 부착 → iMonnit 등록(지시 전 전원 켜지 않음) → 게이트웨이 통신 확인·장착 → 전원 분리 확인 후 릴레이를 제어 장치에 연결(릴레이는 전원선 inline 스위치, 기본 NORMALLY OPEN, iMonnit 편집 페이지에서 NORMALLY CLOSED로 변경 가능, 전원·접지에 연결 안 함) → 전원 인가·iMonnit에서 릴레이 활성 조건 설정.

ALTA PAR Light Meter 구성·설치 가이드

Installation Guides

ALTA Wireless PAR(광합성 유효 복사) Light Meter는 식물 광합성에 이상적인 광 파장을 모니터링합니다. blue-enhanced 실리콘 포토다이오드(스펙트럼 389–692nm ±5nm)로 온도 보상된 PPFD(μMol/m²/s) 측정, 하루 적산하여 DLI(μmol/m²/day) 산출. 리드에 M8 6핀 커넥터로 빠른 분리·교체.

주요 설정: Sensor Name, Heartbeat/Aware State Heartbeat, Low Light Threshold(미만 시 No Light 보고), Saturated Light Threshold(초과 시 Saturated Light·aware), Light Threshold Buffer, DLI Reset Time(매일 PAR DLI를 0으로 리셋, 11pm~11:45pm 권장, 일광절약 자동 조정 안 됨), Measurement Interval(PPFD 측정 빈도), PAR DLI Threshold, Enable Temperature Compensation(온도 소자가 베이스에 있어 리드 끝 환경과 다를 수 있음, 20°C±5°C에서는 1% 미만 영향, 빠른 온도 변화 시 Off 권장).

use case별 DLI: Very Low 4 / Low 7 / Moderate 13 / High 30 / Very High 52 / Full 65 mol/m²/day.

Advanced Vibration Meter(AVM) 구성·설치 가이드

Installation Guides

ALTA Wireless Advanced Vibration Meter(AVM)는 리드 끝 밀봉 큐브 내 가속도계로 표면 진동을 측정합니다. Heartbeat마다 64~128개 가속도 샘플을 최소 1회 측정해 보고(첫 64포인트에서 저대역 에너지가 크면 더 낮은 샘플레이트로 2차 측정). 3직교축(x,y,z) 각각 보고:

  1. Vibration — 시간 영역 Peak 가속도, RMS 가속도, RMS 속도, 또는 Peak-to-Peak 변위
  2. Frequency — 대역 내 진동 에너지가 가장 큰 주파수(Hz 또는 RPM)
  3. Crest Factor — Peak 가속도 / RMS 가속도 비율(비기본 진동의 기계 손상 가능성 또는 설정 범위 초과 가속 여부 지시)

또한 Duty Cycle(보고 구간 중 진동이 Sensitivity Threshold를 초과한 시간 비율)과 큐브 내부 전자부 온도 보고. 최대 4200Hz/252,000RPM, 측정 간격 최소 1초.

주요 설정: Vibration Mode(Velocity RMS/Acceleration RMS/Acceleration Peak/Displacement Peak-to-Peak), Vibration Aware Threshold, Vibration Hysteresis, Vibration Sensitivity Threshold(이하 값 무시·Duty Cycle 미산입, 0이면 모든 측정 분석·Duty 항상 100%), Window Function(Rectangular=필터 없음, Hanning=스펙트럼 누설 최소화, 변위 측정에 강력 권장), Accelerometer Range(최대 관측 g, 초과 시 파형 클리핑·Crest Factor 1.41 미만), Measurement Interval, Sample Rate, Frequency Range Min(Sample Rate/256까지)·Max(Sample Rate/2까지), Power Mode(Low=샘플당 1회 평균, Medium=16회, High Performance=최대 128회·전력 최대·노이즈 최소, 800Hz 이상은 기본 High Performance), Data Mode(Most Recent/Average/Maximum). Scale: Hz 또는 RPM.

큐브 장착: 평가 대상 시스템에 단단히(rigid) 장착이 매우 중요(느슨하면 신호 손실·예측 불가 공진·부정확). 나사형 스터드·접착제·양면테이프·자석 베이스 사용, 축 표시 라벨 참조.

기준선(Baseline) 결정: 정상 작동 상태의 평균 판독값. 먼저 시스템의 기본 작동 주파수(RPM) 파악:

  1. Vibration Mode=Acceleration RMS, Power Mode=High Performance, Accelerometer Range=16G 설정.
  2. 예상 작동 주파수가 Frequency Range 중간 바로 아래에 오도록 Sample Rate·Frequency Range 설정.
  3. 가동 중 판독 관찰(여러 회).
  4. Sample Rate 조정·반복하며 Acceleration RMS가 최대인 주파수 탐색(= 기본 작동 주파수).
  5. 기본 주파수를 Frequency Range 하위 1/3에 두기(상위에 하모닉 관찰 여백 확보 — 건강 악화 시 배수 주파수 출현).

Vibration Mode 선택: Acceleration(>1KHz/60,000RPM), Velocity(10Hz~1KHz/600~60,000RPM), Displacement(<20Hz/1200RPM).

노이즈 관리(효과 순): Power Mode High Performance → Sample Rate 감소 → Sensitivity Threshold 증가 → Frequency Range 축소 → Data Mode Average.

문제 해결: Hardware Error(STS 8/9, 케이블을 노이즈원에서 이격), 비정상 진동(단단한 장착 확인), 비정상 Crest Factor(<1.41=Accelerometer Range 낮음·클리핑, 2.5+=노이즈 또는 하모닉), Duty Cycle 항상 0%(ST 너무 높음)/100%(ST 너무 낮거나 0).

최종 수정일: 2025년 1월 27일
