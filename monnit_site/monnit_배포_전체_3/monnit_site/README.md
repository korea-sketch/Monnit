# Monnit Korea 사이트 — 파일 구조 안내

기존 단일 HTML(약 6MB)을 역할별로 분리했습니다. 이제 무엇을 고치든 해당 파일 하나만 열면 됩니다.

## 폴더 구성

```
monnit_site/
├─ index.html   (66 KB)   ← 화면 구조·문구·섹션. 텍스트 수정은 대부분 여기
├─ style.css    (133 KB)  ← 디자인·색상·애니메이션·반응형
├─ app.js       (180 KB)  ← 동작 로직(필터·탭·폼·렌더링)
├─ data.js      (1.5 MB)  ← 지식베이스 215개 문서 + 가이드 콘텐츠
└─ images/      (45개)    ← 로고·인물·배경 등 분리된 이미지
```

## 무엇을 어디서 고치나

| 하고 싶은 일 | 열 파일 |
|---|---|
| Why Monnit 문구, 섹션 제목 등 화면 텍스트 | `index.html` |
| 색·여백·폰트·버튼 모양·애니메이션 | `style.css` |
| 필터/탭 동작, 폼 처리, 데이터 렌더링 방식 | `app.js` |
| 도움말 문서·가이드 내용 추가/수정 | `data.js` |
| 사진 교체 | `images/` 폴더의 해당 파일 교체(같은 이름) 또는 새 파일 후 src 변경 |

## 중요 — 로드 순서

`index.html` 맨 아래에서 **`data.js` → `app.js`** 순서로 불러옵니다.
이 순서를 바꾸지 마세요. (app.js가 data.js의 데이터를 사용합니다.)

## 실행 방법

`index.html`을 더블클릭하면 브라우저에서 그대로 열립니다.
배포 시에는 이 폴더 전체(images 포함)를 통째로 올리면 됩니다.

## 🎁 프로모션 운영 방법

프로모션은 **페이지마다 HTML 파일을 하나씩 만들어** 두고, 시트에는 그 **HTML 경로만 등록**하면 됩니다. 그러면 프로모션 목록 카드가 자동으로 생기고, 카드를 누르면 그 HTML이 화면 위에 떠서 표시됩니다.

### 운영 흐름 (3단계)

**1) 프로모션 HTML 만들기**
`promotions/_TEMPLATE.html`을 복사해서 내용만 바꾸세요. 사이트와 같은 다크 디자인이 이미 적용돼 있습니다.

```
monnit_site/
└─ promotions/
   ├─ _TEMPLATE.html      ← 복사용 견본 (이 파일은 그대로 두기)
   └─ 2026-spring.html    ← 복사해서 만든 실제 프로모션
```
> 견본 안의 "상담 신청하기" 버튼은 `href="/#contact" target="_top"` 으로 되어 있어, 오버레이 안에서도 사이트 상담 페이지로 정상 이동합니다. 그대로 두세요.

**2) 시트 `Promotions` 탭에 등록**
시트 하단에 **`Promotions`** 탭을 만들고 헤더에 아래 열을 넣은 뒤, 한 줄에 프로모션 하나씩 등록합니다. **굵은 두 칸(title, file)만 필수**이고 나머지는 비워도 됩니다.

| **title** | **file** | period | badge | ended | desc | image | order |
|-----------|----------|--------|-------|-------|------|-------|-------|

| 열 | 뜻 | 필수 | 예시 |
|---|---|---|---|
| **title** | 카드 제목 | ✅ | `봄맞이 도입 프로모션` |
| **file** | 프로모션 HTML 경로 | ✅ | `promotions/2026-spring.html` |
| period | 진행 기간 | | `2026.06.01 ~ 08.31` |
| badge | 배지 문구 | | `진행중` / `신규` |
| ended | 종료 여부 | | 비워두거나 `y` (종료 시 회색 배지) |
| desc | 카드 한 줄 설명 | | `신규 도입 시 게이트웨이 비용 지원` |
| image | 카드 썸네일 URL | | `images/promo-1.jpg` |
| order | 정렬 순서(작을수록 먼저) | | `1` |

> `id` 열은 비워두면 제목에서 자동 생성됩니다. (영문 사이트용으로 `title_en`, `desc_en` 열을 추가하면 영문 모드에서 자동 적용)

**3) 에디터로 관리 (시트 직접 편집 대신)**
`/editor` 를 열면 섹션 목록에 **"🎁 프로모션"**이 있습니다. ＋새 항목 → **제목**과 **HTML 경로(file)**만 넣고 **시트에 바로 저장**하면 끝입니다. 미리보기 버튼으로 사이트 반영도 바로 확인됩니다.

### 정리

- 프로모션 **내용**은 → `promotions/` 폴더의 HTML 파일에서
- 프로모션 **노출(목록)**은 → 시트 `Promotions` 탭(제목 + 경로)에서
- 재배포 없이 시트만 수정해도 목록에 바로 반영됩니다. (HTML 파일을 새로 추가할 때만 배포 필요)
