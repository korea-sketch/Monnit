주요 고객

조도·CO₂·온습도를 정밀 관리해야 하는 그로하우스·약용 작물 재배 시설·실내 농장.

그로하우스는 조도·CO₂·온습도가 생육·품질을 좌우합니다. 조명·환기·냉난방 이상이 생육 편차로 이어지지만 수동 점검은 시간대별 변화를 놓치므로, 주요 환경을 자동 기록하고 이상을 알립니다.

현장 과제

그로하우스는 조도·CO₂·온습도 변화가 품질에 직접 영향을 주지만, 시간대별 변화를 수동으로 관리하기 어렵습니다.

  • 환경 변수 관리 부담 — 조도·CO₂·온습도가 서로 영향 주며 품질 좌우
  • 설비 이상 지연 — 조명·환기·냉난방 이상이 작물 스트레스 확대
  • 구역별 편차 — 재배 구역별 환경 차이로 생육 편차 발생
  • 데이터 부족 — 환경 이력 부족 시 생육 문제 원인 분석난

솔루션

조도·CO₂·온습도를 무선 센서로 수집하고 iMonnit에서 구역별 생육 환경 지속 관리.

조도·CO₂·온도·습도 센서를 재배 구역별로 설치해 주요 생육 환경을 자동 수집합니다. iMonnit이 시간대별 변화·구역별 이력을 보여주고 이탈 시 알려, 조명·환기·냉난방을 데이터 기반으로 조정합니다.

도입 절차

PHASE 1
구역 진단 · 조명·공조 운전 편차가 큰 재배 구역을 식별합니다. · 1일
PHASE 2
센서 배치 · 조도·CO₂·온습도 센서를 재배 구역에 설치·연동합니다. · 1~2일
PHASE 3
생육 기준 설정 · 작물 단계별 적정 환경과 설비 이상 알림을 설정합니다. · 3~5일
PHASE 4
시설 확장 · 검증된 구성을 전 재배 구역으로 확장합니다. · 2~4주
