고객 개요

HDC랩스는 HDC그룹 계열의 AIoT 공간 솔루션 기업으로, 스마트홈·스마트빌딩·부동산 종합관리·인테리어·AI 솔루션을 중심으로 사람·공간·기술을 연결하는 스마트 공간 서비스를 제공합니다. 다수 빌딩의 환경·에너지·점유 데이터를 단일 플랫폼으로 통합해 RMS와 연동하는 것이 핵심 과제였습니다.

과제

  • 수많은 빌딩의 HVAC·조명·전력 운영이 빌딩별로 분산되어 통합 분석 불가
  • 공간 점유율이 가시화되지 않아 임차 공간 최적화와 에너지 절감 기회를 상실
  • 중대재해 사고(정전·화재·침수·가스)에 사후 대응만 가능해 임차인 클레임과 수리 비용 증가

솔루션

상업용 빌딩 통합 시설 관리

전 자산의 환경·에너지·고장 여부·점유 데이터를 단일 플랫폼으로 통합

점유율 분석

회의실·층별 공간 활용도를 가시화해 임대 공간 최적화

에너지·HVAC 효율

환경 데이터 기반 HVAC 자동 최적화로 운영비 절감

설비 고장·침수 선제 대응

설비 고장 즉시 대응과 침수·누수 조기 감지로 손해 최소화

도어 접근 통제

공용·전용 공간 접근 이력을 자동 기록

자산 추적·유지보수

설비별 점검·교체 이력을 통합해 예방 정비 효율화
