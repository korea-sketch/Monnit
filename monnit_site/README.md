# Monnit Korea 사이트 — 파일 구조 안내

기존 단일 HTML(약 6MB)을 역할별로 분리했습니다. 이제 무엇을 고치든 해당 파일 하나만 열면 됩니다.

## 폴더 구성

```
monnit_site/
├─ index.html   (66 KB)   ← 화면 구조·문구·섹션. 텍스트 수정은 대부분 여기
├─ style.css    (133 KB)  ← 디자인·색상·애니메이션·반응형
├─ app.js       (180 KB)  ← 동작 로직(필터·탭·폼·렌더링)
├─ data.js      (1.5 MB)  ← 지식베이스 215개 문서 + 가이드 콘텐츠
└─ images/      (45개)    ← 로고·인물·배경 등 분리된 이미지
```

## 무엇을 어디서 고치나

| 하고 싶은 일 | 열 파일 |
|---|---|
| Why Monnit 문구, 섹션 제목 등 화면 텍스트 | `index.html` |
| 색·여백·폰트·버튼 모양·애니메이션 | `style.css` |
| 필터/탭 동작, 폼 처리, 데이터 렌더링 방식 | `app.js` |
| 도움말 문서·가이드 내용 추가/수정 | `data.js` |
| 사진 교체 | `images/` 폴더의 해당 파일 교체(같은 이름) 또는 새 파일 후 src 변경 |

## 중요 — 로드 순서

`index.html` 맨 아래에서 **`data.js` → `app.js`** 순서로 불러옵니다.
이 순서를 바꾸지 마세요. (app.js가 data.js의 데이터를 사용합니다.)

## 실행 방법

`index.html`을 더블클릭하면 브라우저에서 그대로 열립니다.
배포 시에는 이 폴더 전체(images 포함)를 통째로 올리면 됩니다.

## 🎁 프로모션 운영 방법 (HTML 파일 업로드)

프로모션은 **에디터에서 HTML 파일을 직접 올리는 방식**으로 운영합니다. 폴더에 파일을 넣거나 재배포할 필요가 없습니다 — 올린 HTML 내용이 시트에 저장되고, 사이트가 그 내용을 바로 보여줍니다.

### 운영 흐름 (3단계)

**1) 프로모션 HTML 파일 준비**
`promotions/_TEMPLATE.html`(견본)을 컴퓨터로 내려받아 복사한 뒤, 내용만 바꿔 저장하세요. 사이트와 같은 다크 디자인이 이미 적용돼 있습니다.
> 견본의 "상담 신청하기" 버튼은 `href="/#contact" target="_top"` 으로 되어 있어, 오버레이 안에서도 사이트 상담 페이지로 정상 이동합니다. 그대로 두세요.

**2) 에디터에서 업로드**
`/editor` → 섹션에서 **"🎁 프로모션"** 선택 → ＋새 항목 →
- **제목(title)** 입력
- **프로모션 HTML** 칸의 **📄 HTML 파일 선택** 버튼을 눌러 1)에서 만든 .html 파일을 고름 (👁 미리보기로 확인 가능)
- (선택) 기간·배지·설명·썸네일 입력
- **시트에 바로 저장** 클릭

> ⚠️ HTML 내용이 들어간 프로모션은 반드시 초록색 **"시트에 바로 저장"** 버튼으로 저장하세요. (수동 TSV 복사 방식은 HTML의 줄바꿈 때문에 깨집니다 — 에디터가 자동으로 경고해 줍니다.)

**3) 끝**
저장 즉시 사이트 프로모션 목록에 카드가 생기고, 클릭하면 업로드한 HTML이 오버레이로 표시됩니다. 수정도 같은 방식 — 항목을 골라 파일을 다시 올리거나 칸에서 직접 고친 뒤 저장하면 됩니다.

### 시트 열 구성 (참고)

| 열 | 뜻 | 필수 |
|---|---|---|
| **title** | 카드 제목 | ✅ |
| **html** | 프로모션 HTML 내용 (업로드 시 자동 입력) | ✅ |
| period | 진행 기간 | |
| badge | 배지(진행중/신규) | |
| ended | 종료 여부(y) | |
| desc | 카드 한 줄 설명 | |
| image | 카드 썸네일 URL | |
| file | (대안) HTML 파일 경로 — html 비었을 때만 사용 | |
| order | 정렬 순서 | |

> `id`는 비우면 제목에서 자동 생성됩니다. `html` 대신 `file`(폴더에 올린 파일 경로)을 쓰는 옛 방식도 그대로 지원됩니다.
